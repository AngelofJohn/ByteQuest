### Section Guidelines
- **Summary**: Brief overview (skip for reflection entries)
- **What We Built**: Required - the core of each entry
- **Design Decisions**: Include when choices were made
- **Technical Notes**: Include when code details matter
- **Session Stats**: Optional table of metrics
- **What's Next**: Optional - useful for continuity
- **Closing**: Italicized thought or quote (no "Quote of the Day" label)
